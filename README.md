### compatibility

| client library version | HStream server version     |
| ---------------------- | -------------------------- |
| `v0.1.*`               | >= `v0.9.4` && <= `v0.9.6` |
| `v0.2.*`               | >= `v0.9.4` && <= `v0.9.6` |
